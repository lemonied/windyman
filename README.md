# windyman

React 移动端 UI 框架

项目开发使用react版本
```json
{
  "react": "^16.13.1",
  "react-dom": "^16.13.1",
  "react-router-dom": "^5.2.0"
}
```

## 安装
使用npm
```bash
npm i windyman
```
使用yarn
```bash
yarn add windyman
```
组件样式使用scss，使用前请先安装node-sass
```bash
npm i node-sass --save-dev
```
or
```bash
yarn add node-sass -D
```
